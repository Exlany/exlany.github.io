优化本地站内搜索功能

2016.10.22 Lany 前端    

主题使用的本地搜索功能，需安装hexo-generator-search插件生成 XML 格式的索引文件，或者安装hexo-search插件生成 json 格式的索引文件，本地站内搜索都是基于索引文件的。

但是默认生成search.xml索引文件比较大，这里我们说下优化search.xml索引文件大小，加快搜索速度。
